# 🤝 Contributing to Deep Space Workstation

Thank you for checking out the project! We love community pull requests.

## 🚀 Local Development Setup
To modify the workstation safely, use this exact sandbox workflow:

1. **Fork** this repository to your own GitHub account.
2. **Clone** it locally: `git clone https://github.com/YOUR_GITHUB_USERNAME/deep-space-workstation.git`
3. **Navigate** into the folder: `cd deep-space-workstation`
4. **Build the Environment**: `python -m venv exo_env`
5. **Activate**:
   * Windows: `exo_env\Scripts\activate`
6. **Install dependencies**: `pip install -r requirements.txt`

## 📬 Submitting Changes
* Create a branch for your feature (`git checkout -b feature/amazing-new-metric`).
* Commit your changes cleanly.
* Push to your branch and open a **Pull Request** against our master `main` branch.
